# OxbridgeClass
 - 自制昆明理工大学津桥学院课表、考试、成绩相关快速查询
 - 将学校教务系统功能打包为安卓应用，提供便捷的移动端访问方式，包含系统登录、课程查询、考试安排、成绩查询及个人信息管理等核心功能，并优化移动端交互体验
 - **最低支持版本**: Android 12

# 功能
- 基本的系统登录、课程查询、考试安排、成绩查询及个人信息管理

- ### **1.2.1**
- 优化动画内容
- 美化界面
- 修复已知BUG
- 添加课前提醒以及考试提醒
- 添加源网址功能确保在选课时外网无法访问时能够通过内网访问
- 添加自动检查更新

- ### **1.2.0**
- 内嵌浏览器：直接访问教务系统 Web 页面
- 周课表展示：以网格形式展示本周课程
- 考试列表：展示待考科目、时间、地点、座位号及考试形式
- 成绩查询：支持查看“当前学期”和“所有学期”的成绩
- 离线模式：开启/关闭课表、考试、成绩缓存功能
